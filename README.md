# 🏙️ CivicPulse: Smart Governance & Grievance Redressal Platform

CivicPulse is a full-stack, GIS-enabled web application designed to bridge the gap between citizens and municipal authorities. It provides a real-time, transparent platform for reporting infrastructure issues (like potholes, broken streetlights, or water leaks) and tracks them through to resolution.

## 🚀 Key Features
* **Role-Based Access Control:** Dedicated, secure dashboards for Citizens, Municipal Workers, and City Administrators.
* **Real-Time GIS Mapping:** Integration with Leaflet.js to pinpoint the exact latitude and longitude of reported issues.
* **Gamification Engine:** A built-in points system that rewards citizens (+10 points) for verified reports to encourage civic engagement.
* **Traffic-Light Prioritization:** Automated color-coding (Red/Yellow/Green) to triage infrastructure threats based on severity.
* **Transparent Tracking:** Citizens can track the real-time status of their complaints (Pending -> In Progress -> Resolved).

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3 (Glassmorphism UI), Vanilla JavaScript
* **Backend:** Python, Flask Framework
* **Database:** MySQL / PostgreSQL
* **Mapping API:** Leaflet.js
